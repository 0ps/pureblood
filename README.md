> A Penetration Testing Framework created for Hackers / Pentester / Bug Hunter

## Menu
* Web Pentest <br>
 | Banner Grab <br>
 | Whois <br>
 | Traceroute <br>
 | DNS Record <br>

 | Zone Transfer Lookup <br>

 | Subdomain Scan <br>

 | Reverse IP Lookup <br>
 | Subnet Lookup <br>

 | Deface Page

## Installation

> Any Python Version.

```
$ git clone https://github.com/cr4shcod3/pureblood
$ cd pureblood
$ pip install -r requirements.txt
```
